# Calculator

## Description

This is a simple and responsive Calculator built using HTML, CSS, and JavaScript. It can perform basic arithmetic operations with a clean and user-friendly interface.

## Features

* Responsive Design
* Basic Arithmetic Operations
* Clear (C) Button
* Delete (⌫) Button
* Percentage (%) Support
* Simple and Modern UI

## Technologies Used

* HTML5
* CSS3
* JavaScript
