# Tribute Page

A modern and responsive Tribute Page created using HTML, CSS, and JavaScript.

## 📌 Project Overview

This project is dedicated to Dr. A. P. J. Abdul Kalam, the Missile Man of India and the 11th President of India. It presents his introduction, achievements, and an inspirational quote in a clean and responsive design.

## 🚀 Features

- Responsive Design
- Modern User Interface
- Smooth Scrolling
- Scroll Animation
- Interactive Button
- Mobile Friendly
- Clean Code Structure

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Tribute-Page/

│── README.md
└── images/
    └── kalam.jpg
```

## ▶️ How to Run

1. Download or Clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any web browser.
